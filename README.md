# 📊 Spreadsheet-Based Accounting System

Accounting application built using Golang, React, and Google Sheets as a data layer.

## 🚀 Tech Stack
- Golang (Backend Processing)
- React (Frontend UI)
- Google Sheets (Data Source)

## ✨ Features
- Import bank transaction data
- Map transactions into Chart of Accounts (COA)
- Generate financial reports (General Ledger, Income Statement, Balance Sheet)
- Compare multiple data sources
- Export reports

## 🧠 Highlights
- Complex data processing from spreadsheet
- Financial data transformation pipeline
- Lightweight accounting system (low infrastructure)

## 📂 Project Structure
- backend logic in Golang (.go files)
- frontend UI in /frontend
- data processing & helpers
